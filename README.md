# Portfolio Musa Arradi

Portfolio personal de Musa Arradi, especialista en Technical SEO, AI Search y automatización. Web estática construida para presentar experiencia profesional, proyectos y perfil técnico de forma clara y directa.

**Demo:** [portfolio-musa-phi.vercel.app](https://portfolio-musa-phi.vercel.app)

---

## Estructura del proyecto

| Archivo | Función |
|---------|---------|
| `index.html` | Página principal: hero, experiencia, proyectos destacados, stack y contacto |
| `projects.html` | Casos de uso detallados con sidebar y case studies |
| `blog.html` | Blog con artículos de SEO y AI Search (en preparación) |
| `styles.css` | Design system compartido: tokens, componentes y layout |
| `profile.webp` | Foto de perfil usada en el hero |
| `favicon.svg` | Favicon principal (SVG, todos los browsers modernos) |
| `favicon.png` | Favicon alternativo (PNG fallback) |
| `apple-touch-icon.png` | Icono para iOS / web app |
| `robots.txt` | Permite rastreo completo + referencia al sitemap |
| `sitemap.xml` | Mapa del sitio para indexación en GSC |

---

## Features

- Diseño responsive (mobile, tablet, desktop)
- Páginas de proyectos con case studies detallados
- Blog con filtros por categoría
- Open Graph y Twitter Card en las 3 páginas
- Favicon y apple-touch-icon
- `robots.txt` y `sitemap.xml` incluidos
- Consistencia visual multi-página con design system compartido
- Animaciones fade-in con IntersectionObserver
- Navbar fija con blur y hamburger menu mobile

---

## Stack

- HTML5
- CSS3 (vanilla, sin frameworks)
- JavaScript vanilla
- [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- Deploy en [Vercel](https://vercel.com)
